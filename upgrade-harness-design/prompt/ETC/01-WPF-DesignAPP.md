이 프로젝트는 wpf-xaml을 수집해 애니메이션을 웹으로 창조하는 역할을 가지고 있습니다.

수집된 wpf호환 xaml을 
Belnd for Visual Studio 에서도 로드를해 디자인을 보고, 미세 수정을 하고자합니다.

목적 : 수집한 애니메이션의 효과를 재탄생

## 예시
- design-wpf-app : 개선 프로젝트공간 ( 빈프로젝트로 작성됨 닷넷솔루션/프로젝트참고)
- 수집된 wpf-xaml : design/xaml/sample

## 요청지침
- 좌측패널은 넘버링된 xaml의 네비게이션 메뉴
- 패널클릭시 넘버링된 xaml 컴포넌트를 클릭해서 우측패널에서 보기가능
- 해당 공간에 작성된 원본파일은 건들지말고, design-wpf-app 하위에 복사호환(싱크) 시켜주세요
  - 원본을 그대로 이용해 표현가능하면 좋겠지만.. 호환가능모드로 가져올것
- 내장DB이용해 xaml 마이그레이션 현황을 잘유지할것
  - design-wpf-app 하위에 마이그레이션된 xaml을 관리하는 db 폴더를 하나 만들어서 관리할것
  - 마이그레이션된 xaml은 넘버링된 파일명으로 관리할것 (예시 design-wpf-app/migrated/01-기법이름.xaml)
  - 마이그레이션된 xaml의 상태(미완성,완성,테스트완료 등)도 관리할것
  - 이 디비는 지속업그레이드 될수 있기때문에 이 DB자체의 기능 마이그레이션을 채택할것
