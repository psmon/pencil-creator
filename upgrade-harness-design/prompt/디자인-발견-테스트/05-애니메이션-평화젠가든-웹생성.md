평화로운 젠가든 wpf템플릿 이용해 , 전쟁을 멈추고 평화를 기원하는 홈페이지 만들기

🌿 평화를 기원하는 홈페이지 구성
1. 메인 화면 (Hero Section)
🎯 목적
첫 화면에서 감정 전달 + 메시지 각인

✍️ 문구 예시

헤드라인
“우리는 평화를 선택합니다”
서브 메시지
작은 바람이 모여 세상을 바꿉니다
당신의 마음이 평화의 시작입니다

버튼
“함께 기원하기”
“나의 메시지 남기기”

2. 평화 메시지 섹션
🎯 목적

공감 + 다양한 시각 전달
✍️ 문구
제목:
“평화는 멀리 있지 않습니다”
내용:
서로를 이해하려는 작은 노력
다름을 인정하는 따뜻한 시선
그것이 바로 평화입니다

4. 평화 지도 (옵션)
🎯 목적
“우리가 연결되어 있다”는 시각화
✍️ 문구
전 세계에서 보내온 평화의 메시지
우리는 하나로 연결되어 있습니다
5. 마무리 (Footer 메시지)
✍️ 문구

평화는 거창한 것이 아닙니다
오늘, 당신의 마음에서 시작됩니다
🌱 전체 톤 & 디자인 방향
🎨 디자인 키워드
색상:
화이트, 연한 블루, 그린
분위기:
부드러움 / 여백 / 따뜻함
폰트:
둥글고 읽기 편한 글씨
💡 핵심 메시지 요약

👉 이 홈페이지의 중심 문장:

“평화는 우리가 선택하는 것입니다”

👉 사용자에게 전달할 감정:

희망
연결감
참여 의지
