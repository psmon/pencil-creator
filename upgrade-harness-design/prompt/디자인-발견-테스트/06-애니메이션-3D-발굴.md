WPF 기반 애니메이션 UI를 Space / Astral 테마로 설계 및 조사

단순한 2D UI가 아니라, 깊이감(Z축)과 공간감을 강조한
3D 인터페이스 스타일을 중심으로 분석 및 템플릿 보강

## 핵심 방향
- 우주/은하/성운 기반의 공간형 UI
- 무한 깊이감을 느낄 수 있는 레이어 구조
- 카메라 이동 기반 인터페이스 연출
- WPF에서 구현 가능한 수준으로 구조화

- infinite space illusion

### 2. 카메라 & 시점

- perspective UI transition

### 3. 파티클 & 우주 요소
- starfield particle system
- nebula / cosmic fog effect (pseudo volumetric)

- floating light particles

### 4. 빛 & 볼륨 효과
- volumetric light simulation (fake 방식 포함)
- glow bloom animation

- depth-based opacity & blur

### 5. UI 구성 요소
- floating hologram panels (space에 떠있는 UI)
- circular / radial interface

- orbital UI elements (회전하는 UI)

## WPF 구현 제한 고려 (중요)

아래 조건을 반드시 반영하여 설계:

- Viewport3D 또는 2.5D 방식 혼합
- TranslateTransform / ScaleTransform 기반 depth 연출
- Parallax = 레이어별 속도 차이로 구현
- BlurEffect + Opacity → Depth of Field 표현
- Particle = Canvas + 랜덤 이동 또는 CompositionTarget.Rendering
- ShaderEffect 가능 시 glow/blur 확장
