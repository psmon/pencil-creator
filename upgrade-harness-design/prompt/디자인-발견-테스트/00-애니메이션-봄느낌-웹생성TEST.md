사용스킬 : harness-usage

프롬프트 :
```
방금작성한 wpf-봄시즌 애니메이션을 활용해 

봄애니메이션을 적극 활용하는 바닐라 JS를 웹페이지를만들어죠
- 룩앤필은 화사한 봄느낌, 홈페이지 테마가 있다기보다 애니메이션 효과를 제대로 구현할수 있나? 웹애니메이션 플레이 그라운드의 장임 , 룩앤필 일관성은 신경쓸것
- js,css 분리구현
- js채택은 애니메이션 정의를 살펴본후 최적의 것을 채택
- 이펙트가 필요한경우 이미지파일(png와같은) 말고  BlurHash,SVG,Base64 등 생성해서 활용가능 (단 애니메이션에 어울리고 필요한경우 채택)

작성경로 : design/xaml/output/sample00/

```
