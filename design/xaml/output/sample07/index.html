SECTION · 01

디자인 시스템을 소개합니다

React 18+/19 · TypeScript · Tailwind CSS 4 · Radix UI · shadcn/ui 기반.
내부 팀의 일관된 UI 개발을 위한 사내 맞춤형 컴포넌트 라이브러리입니다.

01

토큰 중심 설계

색상·타이포·간격을 CSS 변수로 정의. 4가지 테마(Light-A, Dark, Theme-B Light/Dark) 간 자동 동기화.

bg-default text-subtle border-default
02

Radix UI 기반 접근성

14개 프리미티브, 키보드 네비, ARIA 속성, 포커스 트랩, 스크린리더 호환까지 내장.

WCAG 2.1 AA
03

설계 소스 단일화

Figma 킷(SortUI)의 시각 정의 + shadcn/ui의 인터랙션 로직을 통합. 디자인 충실도와 생산성을 동시에 확보.

SortUI + shadcn/ui
04

서브패스 임포트

@blumnai-studio/blumnai-design-system/button 형태로 컴포넌트 단위 로드. 번들 크기 최적화.

Bundle Size
-65%
05

엔터프라이즈 복합 컴포넌트

DataGrid, HtmlEditor(Tiptap), EventCalendar, ResizablePanel, DnD(@dnd-kit) 등 복잡한 인터랙션을 기본 제공.

DataGrid Editor Calendar +12
06

점진적 마이그레이션

기존 프로젝트에서 컴포넌트 단위로 점진 도입 가능. Next.js 13+, Vite, webpack 5+ 지원.
