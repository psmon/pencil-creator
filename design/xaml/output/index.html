Case W · sample00

봄 애니메이션 플레이그라운드

방금 작성한 wpf-봄시즌 애니메이션을 활용해 봄 애니메이션을 적극 활용하는 바닐라 JS 웹페이지를 만들어줘 → sample00/ · 벚꽃·바람·bloom
Case W · sample01

사계절 종합 쇼케이스

지금까지 보유한 wpf-애니메이션 템플릿을 모두 활용해 선보이는 PlayGround 페이지를 만들고 싶습니다 룩앤필은 봄·여름·가을·겨울 균형있게 → sample01/ · CAT1~12 전체 활용
Case W · sample02

AI Voice 컴포넌트

음성파동 통합 컴포넌트를 펜슬-wpf-애니메이션을 활용해 구현 고객 챗봇·상담사 AI 콘솔·데이터센터 다이어그램 → sample02/ · SVG Voice Wave
Case W · sample09

픽셀 보이스 랩

sample02를 참고해 PixiJS 활용 CSS 카드가 아닌 픽셀 요소로 UI 재현 AI 보이스 컴포넌트를 픽셀 아트 버전으로 변형 → sample09/ · PixiJS Pixel UI
Case W · sample03

사이버펑크 HUD 패널

Case A+W · sample05

Astral Voyager 우주 쇼케이스

Case A+W · sample04

평화 기원 홈페이지

Case W · sample06

요르단의 반지 RPG 랜딩
