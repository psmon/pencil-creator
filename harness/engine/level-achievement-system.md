# level-achievement-system.md — MMORPG 레벨 & 업적 시스템

모든 글작성(Type A~F) 완료 시 **recording** 단계에서 자동 실행된다.
content-evaluator가 5축 평가 직후 경험치 부여 → 레벨업 판정 → 업적 갱신을 수행한다.

---

## 1. 트리거

```
evaluating 단계 완료 → recording 단계 진입 시:
  1. 5축 평가 점수 확정
  2. 경험치 계산 & 부여
  3. 레벨업 판정
  4. 업적 갱신
  5. 결과를 사용자에게 RPG 스타일로 출력
```

---

## 2. 경험치(XP) 획득 공식

```
획득XP = 기본XP × 등급배율 × 유형배율

기본XP = 평가점수 × 10
  예) 85점 → 850 기본XP

등급배율 (writing-craft.md 등급 기준):
  A등급 (80~100점): ×5  — 유니크 몬스터
  B등급 (60~79점):  ×3  — 챔피언 몬스터
  C등급 (40~59점):  ×1  — 일반 몬스터
  D등급 (0~39점):   ×0.5 — 약한 몬스터

유형배율:
  Type C (양 매체 동시): ×2.0  — 듀얼 보스
  Type D (기존 개선):    ×1.5  — 정예 도전
  Type F (웹노리 위키):  ×1.2  — 탐험 보너스
  Type A/B/E:            ×1.0  — 기본
```

### 계산 예시

| 상황 | 점수 | 등급 | 유형 | 기본XP | 등급배율 | 유형배율 | **최종XP** |
|------|------|------|------|--------|---------|---------|-----------|
| 최고 A등급 Memorizer | 95 | A | A | 950 | ×5 | ×1.0 | **4,750** |
| 평균 B등급 Confluence | 72 | B | B | 720 | ×3 | ×1.0 | **2,160** |
| 양 매체 A등급 | 88 | A | C | 880 | ×5 | ×2.0 | **8,800** |
| 개선 작업 B등급 | 75 | B | D | 750 | ×3 | ×1.5 | **3,375** |
| 웹노리 A등급 | 91 | A | F | 910 | ×5 | ×1.2 | **5,460** |
| 저점수 D등급 | 30 | D | A | 300 | ×0.5 | ×1.0 | **150** |

## 3. 레벨 시스템 (Lv.1 ~ Lv.100)

### 3.1 필요 경험치 공식

레벨 L에서 L+1로 올리는 데 필요한 XP:

```
구간별 필요XP(L → L+1):

Lv. 1~10  (견습 작가):    XP(L) = 100 + 50 × L
Lv. 11~50 (숙련 작가):    XP(L) = 300 + 100 × (L - 10)
Lv. 51~70 (전문 작가):    XP(L) = 4,300 + 300 × (L - 50)
Lv. 71~90 (마스터 작가):  XP(L) = 10,300 + 800 × (L - 70)
Lv. 91~99 (전설의 작가):  XP(L) = 26,300 + 3,000 × (L - 90)
```

### 3.2 주요 레벨 필요 XP 참조표

| 레벨 | 다음 레벨 필요XP | 누적 총 XP (대략) | 칭호 |
|------|----------------|-----------------|------|
| 1→2 | 150 | 150 | 펜 잡은 모험가 |
| 5→6 | 350 | 1,250 | 잉크 냄새를 아는 자 |
| 10→11 | 600 | 3,150 | 견습 작가 |
| 25→26 | 1,800 | 18,900 | 키보드 워리어 |
| 50→51 | 4,300 | 92,850 | 숙련 작가 |
| 70→71 | 10,300 | 231,550 | 전문 작가 |
| 80→81 | 18,300 | 374,550 | 콘텐츠 마에스트로 |
| 90→91 | 26,300 | 558,550 | 마스터 작가 |
| 95→96 | 41,300 | 727,550 | 대현자의 깃펜 |
| 99→100 | 53,300 | 916,750 | 전설의 작가 |

### 3.3 레벨 구간별 칭호

| 구간 | 칭호 | 설명 |
|------|------|------|
| 1~5 | 펜 잡은 모험가 | 글쓰기 세계에 첫 발을 내딛다 |
| 6~10 | 잉크 냄새를 아는 자 | 이제 좀 감이 오기 시작 |
| 11~25 | 키보드 워리어 | 꾸준히 글을 생산하는 전사 |
| 26~50 | 숙련 작가 | 하네스가 몸에 배기 시작 |
| 51~70 | 전문 작가 | 5축 평가가 두렵지 않은 경지 |
| 71~80 | 콘텐츠 마에스트로 | 어떤 매체든 자유자재 |
| 81~90 | 마스터 작가 | 하네스를 역으로 가르치는 자 |
| 91~95 | 대현자의 깃펜 | 한 글자에 경험치가 녹아든다 |
| 96~99 | 전설 그 자체 | 더 이상 증명할 것이 없는 자 |
| 100 | BloomLabs의 신화 | 만렙. 당신이 곧 하네스다 |

## 4. 업적(Achievement) 시스템

### 4.1 업적 유형

#### A. 카테고리 최초 도전 업적

콘텐츠 주제를 카테고리로 분류하고, **해당 카테고리의 첫 글 작성 시** 업적 달성.

카테고리 목록 (content-domain.md 도메인 기반 + 확장 가능):

| 카테고리 | 첫 도전 업적명 | 패러디 원본 |
|----------|--------------|-------------|
| AI/LLM | "나는 프롬프트한다 고로 존재한다" | 데카르트 "나는 생각한다 고로 존재한다" |
| 개발방법론 | "집밥 아키텍트" | tvN 「집밥 백선생」 |
| 인프라/DevOps | "서버와 바다" | 헤밍웨이 「노인과 바다」 |
| QA/테스트 | "버그도 너무 버그" | "마나도 너무 마나" |
| 프론트엔드 | "빛으로 렌더얍!" | 세일러문 변신 대사 |
| 백엔드 | "개인적인 감정은 없어요, 500 에러지만" | 오버워치 메이 대사 |
| 데이터/분석 | "해저 2만로우" | 쥘 베른 「해저 2만리」 |
| 보안 | "작전명 패치워크" | 영화 「작전명 발키리」 |
| 제품기획 | "가슴이 웅장해지는 PRD" | 인터넷 밈 "가웅" |
| 도구/생산성 | "만능 천재" | 마비노기 타이틀 |
| 기타/에세이 | "평범한 작가인 내가 알고보니 하네스 최강자?" | 이세계물 라노벨 제목 패러디 |

#### B. 카테고리 반복 업적

동일 카테고리에서 글을 반복 작성하면 단계별 업적 달성:

| 횟수 | 업적 접미사 | 예시 (AI/LLM 카테고리) |
|------|-----------|----------------------|
| 10회 | "~의 단골손님" | "AI/LLM의 단골손님" |
| 50회 | "~에 미친 자" | "AI/LLM에 미친 자" |
| 100회 | "~을 지배하는 자" | "AI/LLM을 지배하는 자" |

#### C. 특별 업적 (숨겨진 업적)

| 조건 | 업적명 | 패러디 |
|------|--------|--------|
| 첫 글 작성 완료 | "펜을 잡은 순간" | - |
| A등급 첫 달성 | "넌 내게 A를 줬어" | "넌 내게 반했어" |
| 5축 만점 (100점) | "가슴이 웅장해진다" | 인터넷 밈 |
| Type C 첫 달성 | "듀얼 와일드" | D&D 이도류 |
| Type D로 20점 이상 개선 | "죽은 척의 달인" | 마비노기 타이틀 (부활) |
| 연속 3회 A등급 | "유혹을 이겨낸 자" | 마비노기 (고품질 유지의 유혹) |
| 10종 카테고리 최초 도전 달성 | "슈퍼스타" | 마비노기 타이틀 |
| 레벨 50 도달 | "참을 수 없는 글쓰기" | 쿤데라 소설 패러디 |
| 레벨 100 도달 | "BloomLabs의 신화가 된 자" | - |

## 5. 퍼시스턴스 (저장소 구조)

### 5.1 레벨 상태: `harness/logs/level-up/status.json`

```json
{
  "current_level": 1,
  "current_xp": 0,
  "xp_to_next_level": 150,
  "total_xp_earned": 0,
  "title": "펜 잡은 모험가",
  "total_articles": 0,
  "last_updated": "yyyy-mm-dd"
}
```

### 5.2 레벨업 히스토리: `harness/logs/level-up/history.md`

레벨업 발생 시마다 1줄 추가:

```markdown
| 날짜 | 레벨 | 획득XP | 트리거 콘텐츠 | 칭호 |
```

### 5.3 업적 상태: `harness/logs/achievements/achievements.json`

```json
{
  "categories": {
    "AI/LLM": { "count": 0, "first_date": null, "milestones": [] },
    ...
  },
  "special": [],
  "last_updated": "yyyy-mm-dd"
}
```

### 5.4 업적 로그: `harness/logs/achievements/history.md`

업적 달성 시마다 1줄 추가:

```markdown
| 날짜 | 업적명 | 유형 | 카테고리 | 트리거 콘텐츠 |
```

  ████████░░░░░░░░ 2,450 / 4,500 XP

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
