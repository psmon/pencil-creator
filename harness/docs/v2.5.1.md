# v2.5.1 — Playwright 자동검수 제거 (요청 시에만 실행)

**날짜**: 2026-03-31
**버전 범프**: 2.5.0 → 2.5.1 (패치)

---

## 변경 동기

Playwright 기반 스크린샷 캡처 및 browser_evaluate 분석이 모든 Case C/W 워크플로우에서 **자동으로 실행**되어 불필요한 오버헤드를 유발.
사용자가 명시적으로 요청한 경우에만 Playwright를 활용하도록 정책 변경.

---

## 변경 내역

### Layer 1 — Knowledge (design-craft.md)

| 항목 | Before | After |
|------|--------|-------|
| Case C 핵심 원칙 | WebFetch+Playwright 필수 | WebFetch 중심, Playwright는 사용자 요청 시 보조 |
| C1 25점 기준 | "WebFetch + Playwright 병행" | "WebFetch 심층 분석" |
| C1 판정 방법 | Playwright 활용 여부가 필수 체크 항목 | WebFetch 분석 충분성 중심, Playwright는 가점 |

### Layer 2 — Agents (design-evaluator.md)
