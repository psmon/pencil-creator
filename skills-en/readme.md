# Skills (English Translation)

These skills were originally developed in Korean and have been translated to English for international users.

**This is a translated version only** — automated installation is not supported.
To use these English skills, manually copy the desired skill folders into your `.claude/skills/` directory.

```
# Example: copy all English skills
cp -r skills-en/harness-creator/  .claude/skills/harness-creator/
cp -r skills-en/harness-usage/    .claude/skills/harness-usage/
cp -r skills-en/pencil-design/    .claude/skills/pencil-design/
cp -r skills-en/pencil-deploy/    .claude/skills/pencil-deploy/
```

> The original Korean skills live in `.claude/skills/` and remain the primary source of truth.
